# Правда или действие 18+

Статическое мини-приложение для взрослой компании: 2-4 игрока, имена и аватары,
ход по кругу, выбор "Правда" или "Действие", ограничение на две правды подряд.

## Что внутри

- `index.html` — разметка экранов.
- `styles.css` — мобильный экстравагантный дизайн под формат VK Mini App.
- `app.js` — логика игры и база из 300 вопросов + 300 действий.

## Как открыть

Можно открыть через любой локальный сервер из папки проекта:

```bash
python -m http.server 8080
```

После этого перейти на `http://localhost:8080`.

Для VK Mini Apps проект можно разместить как статический сайт на VK Hosting или
другом HTTPS-хостинге. Если в окружении будет доступен `vkBridge`, приложение
вызовет инициализацию `VKWebAppInit`.
